# Spotify Artist Network Analysis

## Overview
This project analyzes relationships between Spotify artists using PySpark and NetworkX.

## Dataset
- 150,000+ Spotify artists
- 300,000+ artist relationships

## Tools & Technologies
- Python
- PySpark
- NetworkX
- Jupyter Notebook

## Analysis Performed
- Graph creation
- Centrality analysis
- BFS sampling
- Follower prediction

## Key Results
- Graph with 156,326 nodes
- 300,386 edges analyzed
- Linear Regression model for follower prediction

## Skills Demonstrated
- Data Analysis
- Big Data Processing
- Graph Analytics
- Machine Learning
